## Docker Setup
```
docker-compose up
```

* O comando subirá o container com o banco da aplicação (Postgres) - porta 5432
* Junto com um interface (pgAdmin 4) - porta 16543


## API Setup
```
yarn dev:sever
```

* Para subir o serviço em modo de desenvolvimento - porta 3333
